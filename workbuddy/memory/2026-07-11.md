# 2026-07-11 工作日志

## 逆战未来 · 武器伤害计算器（无限画布节点编辑器）

### 完成内容
- 创建了基于无限画布的节点式武器伤害计算器
- 技术栈: 纯 HTML + CSS + JavaScript (无框架依赖)
- 文件结构:
  - `data.js` — 武器数据（主武器9类、副武器3类、近战武器、插件20种）
  - `index.html` — 页面结构
  - `styles.css` — 暗色游戏主题样式
  - `app.js` — 核心逻辑（画布平移缩放、节点拖拽、连线、DPS计算引擎）

### 核心功能
1. **无限画布**: 支持鼠标拖拽平移、滚轮缩放(0.3x~2.5x)、网格背景
2. **5种节点类型**: 武器输入(青色)、插件(橙色)、副武器(紫色)、近战武器(绿色)、输出结果(黄色)
3. **节点连线**: 端口拖拽创建贝塞尔曲线连线，支持环检测、删除
4. **DPS计算引擎**: 递归遍历节点图，按连线顺序叠加插件效果
   - DPS = (基础伤害 × 倍率 + 固定加成) × 射速/60 × (1 + 暴击率 × (暴击伤害-1))
5. **右侧筛选面板**: 4个标签(主武器/副武器/近战/插件)，主武器9子分类，副武器3子分类，搜索功能
6. **初始示例**: 自动创建 AK-47 → 穿甲弹 → 输出 的示例链路

### 武器分类
- 主武器: 突击步枪、狙击步枪、霰弹枪、火箭发射器、冲锋枪、机枪、射手步枪、连发榴弹、喷射器
- 副武器: 手枪、单发榴弹、弓箭
- 近战武器: 8种
- 插件: 20种（含倍率/固定/暴击等多种修饰类型）
